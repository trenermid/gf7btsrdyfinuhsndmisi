## 📌 О проекте

Данный проект представляет собой интерактивную 3D-симуляцию **закона всемирного тяготения**, разработанную в игровом движке Unity. Проект наглядно демонстрирует, как массивное тело (планета) притягивает к себе менее массивный объект (спутник) в соответствии с формулой:

> **F = G × (M × m) / r²**

где:
- `F` — сила гравитационного притяжения
- `G` — гравитационная постоянная (упрощённая для наглядности)
- `M` — масса планеты
- `m` — масса спутника
- `r` — расстояние между объектами

## 🎮 Управление

| Клавиша | Действие | Что происходит |
|:-------:|----------|----------------|
| **↑** | Увеличить гравитацию | Спутник притягивается сильнее |
| **↓** | Уменьшить гравитацию | Спутник притягивается слабее |
| **M** | Увеличить массу спутника | Объект становится больше и тяжелее |
| **N** | Уменьшить массу спутника | Объект становится меньше и легче |
| **Пробел** | Сбросить позицию спутника | Спутник возвращается на стартовую позицию |
| **R** | Полный сброс | Сбрасываются все параметры (G=5, масса=1) |

## 🎯 Демонстрируемые физические явления

| Визуальный элемент | Физическое явление |
|--------------------|--------------------|
| 🟠 Оранжевая планета | Массивное тело, создающее гравитационное поле |
| ⚪ Серый спутник | Пробное тело, движущееся под действием гравитации |
| 🟡➡️🔴 Жёлто-красная линия | Траектория движения спутника |
| 💡 Краснеющий свет | Увеличение силы притяжения при приближении |
| 🔴 Покраснение спутника | Нагрев тела при падении (переход энергии) |
| 📏 Изменение размера планеты | Визуализация изменения массы |
